# Bazaar-Information-Gathering-666

A web application for gathering information from social media and other sources, with AI assistance.

## Features

- Contact form with phone number
- Web crawling using requests and BeautifulSoup
- Social media crawling using Selenium for dynamic content
- AI assistant powered by OpenAI
- Automatic internet connection for data fetching

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Set OpenAI API key: `export OPENAI_API_KEY=your-key`
3. Run: `python app.py`

## Usage

- Visit http://localhost:5000
- Use the contact form to submit information
- Enter a URL to crawl web data
- Ask questions to the AI assistant
